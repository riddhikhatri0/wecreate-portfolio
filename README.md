# WeCreate 🎨 — Portfolio Website

A retro pixel-art styled portfolio website for **WeCreate**, a freelance creative duo (Riddhi + Dhruv) offering graphic design, illustration, branding, and UI/UX services.

🔗 **Live Site:** https://wecreate-portfolio-eight.vercel.app/

## ✨ Features

- **Dynamic time-based backgrounds** — the scene (sky, sun/moon, lighting) changes based on the time of day (morning, afternoon, evening, night)
- **Retro CRT/TV-style interface** — nostalgic pixel-art aesthetic with a desk, plant, and framed art
- **Tab-based navigation** — About, Services, Our Work, and Connect sections styled as TV channel buttons
- **About Us** — intro to the duo, their story, and skills/tools (Figma, Photoshop, Illustrator, Canva Pro, HTML/CSS, UI/UX)
- **Services** — overview of offerings: website design, Instagram posts, flyers & banners, business cards, posters, and brand identity
- **Our Work** — interactive portfolio gallery; click projects to reveal/expand case studies
- **Connect** — contact form with name, email, and message fields, plus direct WhatsApp links, email, and Instagram handle

## 🛠️ Tech Stack

- **React** (Create React App)
- Deployed on **Vercel**
